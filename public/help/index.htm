Blank News

Приложение для чтения новостей из RSS-лент с группировкой похожих материалов.

Главная

Отображается лента новостей из настроенных источников. Кнопка обновления в шапке подгружает данные заново.

Контекстная справка

Нажмите F1 на любой экране приложения — откроется раздел справки, соответствующий этой странице.
